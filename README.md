
# Preprocesamiento de Ciencia de Datos

## Objetivo
Aplicar técnicas de preprocesamiento de datos utilizando Python y Pandas, incluyendo manejo de valores nulos, normalización, codificación de variables categóricas y eliminación de duplicados.
