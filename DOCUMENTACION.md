
# Documentación del Proyecto

## Descripción
Este proyecto implementa un proceso básico de preprocesamiento de datos utilizando Python y Pandas.

## Actividades realizadas

- Creación de repositorio en GitHub
- Configuración de Git y GitHub Desktop
- Uso de ramas
- Creación de commits
- Eliminación de valores nulos
- Eliminación de registros duplicados

## Herramientas utilizadas

- Python
- Pandas
- Git
- GitHub
- Visual Studio Code

## Resultados

El script me permitió limpiar el conjunto de datos eliminando información incompleta y registros duplicados.
